# Fifty Flowers — Senior Full Stack Technical Test (test-3)

This repository is the submission target for the Senior Full Stack Developer technical test at Fifty Flowers.

Read the full requirements in [`Technical_Test.md`](./Technical_Test.md).

## How to submit

1. Fork or create your own implementation repository.
2. Open a Pull Request against this repo with:
   - Your implementation as the diff
   - A `README.md` explaining setup, scripts, and your technical decisions
   - A demo (video or deployed link)
3. Estimated effort: ~3.5h required, +1h if you tackle bonuses.

Bonuses are optional and do not affect the baseline evaluation. They are valued as positive signals of Senior-level depth.
